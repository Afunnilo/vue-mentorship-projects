# Startup Landing Page

- CSS
- Fontawesome icons

## Live Demo

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

yarn, vue CLI, node

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```
